* **HTML5** — Semantik va tizimli tuzilma uchun.

* **Box Model:** Bloklar orasidagi masofalarni aniq hisoblash uchun `margin` va `padding`.
* **Centering:** Elementlarni sahifa markaziga keltirish va matnlarni tekislash.
* **UI Komponentlar:**
    * **Profile Card:** Tugmalar va "tag"lar (badge) bilan ishlash.
    * **Pricing Cards:** Uch xil tarif rejasini (`Starter`, `Pro`, `Team`) vizual ajratish.
* **Interactive Elements:** Tugmalarning turli holatlari va foydalanuvchi interfeysi qulayligi.

# IT-Park: HTML and CSS Fundamentals - Project #4 (MiniUI Dashboard)

This is my fourth practical project at **IT-Park**, featuring multiple complex UI blocks: **Navigation Bar**, **Profile Card**, and **Pricing Cards**. The primary focus was on horizontal alignment and mastering the CSS Box Model structure.

## 🛠 Technologies Used
* **HTML5** — For semantic and structural markup.
* **CSS3** — For layout management and advanced design elements.

## 📝 Project Features and CSS Focus
The following technical solutions were implemented in this project:

* **Layout Management:** Using `display: inline-block` and `width` to align elements side-by-side.
* **Box Model Mastery:** Precise spacing calculations using `margin` and `padding`.
* **Centering:** Aligning content and containers to the center of the page.
* **UI Components:**
    * **Profile Card:** Working with buttons and skill badges.
    * **Pricing Cards:** Visually distinguishing three different pricing plans (`Starter`, `Pro`, `Team`).
* **Interactive Elements:** Styling various button states and improving overall UX.
